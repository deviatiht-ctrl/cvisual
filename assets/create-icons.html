Instructions pour créer les icônes PWA

Utilisez votre logo (logo.jpg) et créez les versions suivantes:

  • icon-192.png - 192x192 pixels
  • icon-512.png - 512x512 pixels

Vous pouvez utiliser un outil en ligne comme:

  • RealFaviconGenerator
  • PWA Builder
  • Ou un éditeur d'image pour redimensionner logo.jpg
